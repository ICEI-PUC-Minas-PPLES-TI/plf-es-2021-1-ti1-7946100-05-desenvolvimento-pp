## Instruções de utilização

#### Pré requisitos:
 - node 12.x 

#### Passos:
 - Clonar o repósitorio ou extrair o .zip na máquina local
 - Abrir o prompt de comando no diretório Codigo
 - Executar o comando 'npm i'
 - Executar o comando 'npm run start'
